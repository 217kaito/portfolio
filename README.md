# portfolio

Kaitoのポートフォリオサイト。

- `kaito-portfolio/` — サイト本体（Next.js + TypeScript + Tailwind CSS）。開発・ビルド手順は [kaito-portfolio/README.md](kaito-portfolio/README.md) を参照。
- `.github/workflows/deploy.yml` — mainブランチへのpushでGitHub Pagesへ自動デプロイ。

## 公開手順

1. リポジトリの Settings → Pages → Source を「GitHub Actions」に設定
2. mainブランチにpush（またはdevelopからmainへマージ）
3. 公開用リンクは[こちら](https://217kaito.github.io/portfolio/)
